## Terraform Oracle Cloud Infrastructure Provider Examples 
###  Examples In This Directory 
This directory contains Terraform configuration files showing how to create specific resources. The examples are intended to be as simple as possible, in most cases containing only the specific resource and any dependencies required for it to run. These do not represent production configurations or real world scenarios. The magic buttons of the examples can be found below.
